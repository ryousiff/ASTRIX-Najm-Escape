# 🚀 ASTRIX: Najm Escape

A fast-paced 2D space survival game built in Unity where players navigate through a dangerous cosmic field and try to survive as long as possible.

---

## 🎮 Game Overview

**ASTRIX: Najm Escape** challenges players to control a spacecraft and dodge incoming planetary obstacles. The longer you survive, the higher your score. Precision, timing, and quick reflexes are key to mastering the game.

---

## ✨ Features

- 🚀 Smooth player movement using physics-based controls
- 🪐 Dynamic obstacle avoidance gameplay
- 📈 Real-time scoring system
- 🏆 Persistent high score using PlayerPrefs
- 🔊 Background music and sound effects
- 💥 Explosion effects on collision
- 🔁 Restart system with Game Over UI
- 📱 Mobile-ready input (touch support via Input System)
- 🌐 WebGL build for browser play

## 🧠 Technical Highlights

- Implemented **Input Actions** for cross-platform (mouse + mobile)
- Used **UI Toolkit** instead of traditional Canvas UI
- Managed UI states with **VisualElement display toggling**
- Optimized physics movement with Rigidbody2D
- Clean separation between **Game Logic and UI**
